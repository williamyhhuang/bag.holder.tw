## 🎯 系統特色

### 🏗️ 架構設計
- **微服務架構**: API 伺服器、掃描器、Telegram Bot 分離
- **容器化部署**: Docker + Docker Compose 一鍵部署
- **資源最佳化**: 針對 Mac Mini 2015 的 4GB RAM 最佳化
- **高可用性**: 熔斷器、重試、健康檢查機制

### 📈 技術指標
- **移動平均**: MA5, MA10, MA20, MA60
- **相對強弱指數**: RSI(14)
- **MACD**: 快線、慢線、柱狀圖
- **布林通道**: 上軌、中軌、下軌
- **成交量**: 20 日均量

### 🎯 交易信號
- **黃金交叉**: MA5 突破 MA20
- **死亡交叉**: MA5 跌破 MA20
- **RSI 超賣/超買**: < 30 / > 70
- **MACD 交叉**: 快線穿越信號線
- **布林通道突破**: 價格突破上下軌
- **成交量異常**: 超過平均量 2 倍

### 📱 Telegram 功能
- **即時查詢**: 輸入股票代號即時查價
- **關注管理**: 加入/移除關注股票
- **智能篩選**: 動能股、超賣股、突破股
- **警報推送**: 即時買賣信號通知
- **投資組合**: 持股追蹤與分析

### 💾 資料管理
- **增量更新**: 只更新新的歷史資料
- **自動清理**: 保留 1 年資料，自動清理舊資料
- **備份機制**: 定期資料庫備份
- **錯誤恢復**: 資料損壞自動修復
