這個專案要有以下功能
1. 可以透過指令並指定起迄時間（若沒指定則下載昨天到今天的資料），透過 yfinance 或是 fubon API 下載台股所有上市櫃股票的資料，並儲存成 csv
    - 每個股票代碼一個csv檔
2. 可以透過指令執行台股現貨選股策略，選股策略會讀取所有上市櫃股票的csv檔進行分析，並選出今日適合操作的股票
    - 分析結果需包含：
        - 做多還是做空
        - 欲操作股票代號及名稱
    - 分析結果可以透過 API 傳送到 telegram
    - 使用者可以透過 telegram 回傳其操作的股票及成本，收到後將其記錄至csv中，方便後續分析勝率及損益
        - 使用者會回傳以下資訊：
            - 做多/做空
            - 股票代號
            - 成本
3. 可以透過指令進行回測，回測程式讀取所有上市櫃股票的csv檔進行分析，並將回測結果整理成報告
    - 報告需包含：
        - 選股策略
        - 回測資料起迄時間
        - 勝率
        - 選股策略建議優化方向
4. 可以透過指令執行台股期貨操作建議，並將建議透過 API 傳送至 telegram
    - 期貨資料來源透過 fubon API 取得
    - fubon API 文件
        - https://www.fbs.com.tw/TradeAPI/llms.txt
    - 只分析以下期貨
        - 微台
